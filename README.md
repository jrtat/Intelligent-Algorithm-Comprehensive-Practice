# 知识图谱构建

## 实体

- 岗位（名）
- 待遇（薪资）
- 技能（这个是为了胜任工作所需要的技能）
- 岗位内容（这部分是工作时都在干什么）
- 公司所在地（气候问题）
  - 东部：沿海
  - 西部：非沿海
  - 中部：非沿海
  - 北部：冬季寒冷
  - 南部：潮湿，夏季高温
  - 省-市：一线/新一线/二线/其他
- 公司（名）
- 企业规模
  - 20人以下：微型
  - 20-99人，100-299人：小型
  - 299-499人，500-999人：中小型
  - 1000-9999人，9999人以上：大型
  - 未知
- 企业类型
  - 天使轮/A轮/B轮：公司处于快速上升期
  - C轮...：公司处于平稳上升期
  - 已上市：公司强大，且当前比较稳定
  - 未融资，不需要融资：公司处于平稳期
  - 未知（但企业规模已知）：1000人以上，处于稳定，否则不能确认公司稳定
  - 未知（企业规模未知）：不能确认公司稳定
