# 知识图谱构建

## 实体
- 岗位名称
  - 约100种
- 岗位
  - 约10000个
- 待遇（薪资）
  - 6k以下：低薪
  - 6k-9k：低中薪
  - 9k-12k：中薪
  - 12k-18k：中高薪
  - 18k+：高薪
  - 未知：不允许此处未知
- 技能（这个是为了胜任工作所需要的技能）
  - 要通过某种机器学习算法来实现从"岗位详情"列提取出所有属于该实体的对象
- 学历要求（反映学习能力）
  - 中专及以上
  - 大专及以上
  - 本科及以上
  - 硕士及以上
  - 博士及以上
- 工作经验：
  - 无任何要求
  - 应届生优先
  - 有实习者优先
  - 有工作经验者优先
- 城市
  - 统计excel中的所有的城市
- 城市地理环境
  - 东部：沿海
  - 西部：非沿海
  - 中部：非沿海
  - 北部：冬季寒冷
  - 南部：潮湿，夏季高温
  - 未知：出国等
- 城市社会环境
  - 一线
  - 新一线
  - 二线
  - 二线以下
  - 外派
- 公司（名）
- 行业
  - 通过某种传统算法来实现从"所属行业"列提取出所有属于该实体的对象
- 企业规模
  - 20人以下：微型
  - 20-99人，100-299人：小型
  - 299-499人，500-999人：中小型
  - 1000-9999人，9999人以上：大型
  - 未知
- 企业类型
  - 天使轮/A轮/B轮：公司处于快速上升期
  - C轮...：公司处于平稳上升期
  - 已上市：公司强大，且当前比较稳定
  - 未融资，不需要融资：公司处于平稳期
  - 未知（但企业规模已知）：1000人以上，处于稳定，否则不能确认公司稳定
  - 未知（企业规模未知）：不能确认公司稳定

## 关系
岗位 → REQUIRES → 技能（权重：技能出现频次）
岗位 → OFFERS → 待遇（薪资）
岗位 → BELONGS_TO → 公司
岗位 → LOCATED_IN → 城市 
岗位 → REQUIRES_EDU → 学历要求
岗位 → REQUIRES_EXP → 经验要求
岗位 → TYPE_IS → 岗位名称
公司 → IN_INDUSTRY → 行业
公司 → HAS_SIZE → 企业规模
公司 → HAS_TYPE → 企业类型
城市 → HAS_CLIMATE → 气候区
城市 → HAS_TIER → 城市等级
