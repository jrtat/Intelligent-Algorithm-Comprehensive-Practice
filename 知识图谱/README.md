# 知识图谱构建

## 数据预处理

- 除了薪资，其他列的预处理都保留了原有列

- 学历要求（反映学习能力）预处理后，有以下结果：
  - 维度一：无任何要求 / 中专及以上 / 大专及以上 / 本科及以上 / 硕士及以上 / 博士及以上
  - 维度二：无任何要求 / 双一流 / 985、211

- 城市水平：一线 / 新一线 / 二线 / 二线以下 / 外派

- 企业规模：A：20人以下 / B：20-99人 / C：100-299人 / D：299-499人 / E：500-999人 / F：1000-9999人 / G：10000人以上 / H：未知

- 企业类型
  - 天使轮/A轮/B轮：Soar（高风险高回报，可能称为原始股东）
  - C轮...：公司处于平稳上升期：Rise（中高风险中高回报）
  - 已上市：公司强大，且当前比较稳定：Strong（进大厂）
  - 未融资，不需要融资（企业规模已知为E或F或G）：公司处于平稳期：Stable
  - 未融资，不需要融资（企业规模已知为A或B或C或D）：公司很小，但未必平稳：Unknown
  - 未知（或者未融资，不需要融资但企业规模未知）：这种企业的发展成谜：Unknown

- 创新能力：分4档，大模型打分

- 抗压能力：分4档，大模型打分

- 沟通能力：分4档，大模型打分

- 学习能力：分4档，大模型打分

- 实习能力：分4档，（无 / 6月以下 / 1年以下 / 1年以上）大模型打分

- 职业技能：非结构化文本

- 证书要求：非结构化文本

## 知识图谱构建

### 实体

- 岗位名称（约100种）
  - 工资情况（月薪最小、月薪最大、平均月薪、四分位数）
  - 学历分布
  - 创新，沟通，抗压，能力要求（平均值？）
  - 对实习时间要求的分布
  - 垂直职业发展方向（集合）
  - 城市分布

- 行业
  - 在GB/T 4754-2017所属的门类

- 公司
  - 企业规模
  - 企业类型
  - 公司发展情况
  - 企业描述（非结构化文本）

- 证书

- 技能

### 关系

岗位 --需要--> 技能

岗位 --需要--> 证书

岗位 --涉及--> 企业 

岗位 --涉及--> 行业
