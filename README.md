# 知识图谱构建

## 实体
- 岗位名称
  - 约100种
- 岗位
  - 约10000个
- 待遇（薪资）
  - 6k以下：低薪
  - 6k-9k：低中薪
  - 9k-12k：中薪
  - 12k-18k：中高薪
  - 18k+：高薪
  - 未知：不允许此处未知
- 技能（这个是为了胜任工作所需要的技能）
  - 要通过某种机器学习算法来实现从"岗位详情"列提取出所有属于该实体的对象
- 学历要求（反映学习能力）
  - 中专及以上
  - 大专及以上
  - 本科及以上
  - 硕士及以上
  - 博士及以上
- 工作经验：
  - 无任何要求
  - 应届生优先
  - 有实习者优先
  - 有工作经验者优先
- 城市
  - 统计excel中的所有的城市
- 城市地理环境
  - 东部：沿海
  - 西部：非沿海
  - 中部：非沿海
  - 北部：冬季寒冷
  - 南部：潮湿，夏季高温
  - 未知：出国等
- 城市社会环境
  - 一线
  - 新一线
  - 二线
  - 二线以下
  - 外派
- 公司（名）
- 行业
  - 采用GB/T 4754-2017中的20个门类，97个大类
- 企业规模
  - A：20人以下
  - B：20-99人
  - C：100-299人
  - D：299-499人
  - E：500-999人
  - F：1000-9999人
  - G：10000人以上
  - H：未知
- 企业类型
  - 天使轮/A轮/B轮：Soar（高风险高回报，可能称为原始股东）
  - C轮...：公司处于平稳上升期：Rise（中高风险中高回报）
  - 已上市：公司强大，且当前比较稳定：Strong（进大厂）
  - 未融资，不需要融资（企业规模已知为E或F或G）：公司处于平稳期：Stable
  - 未融资，不需要融资（企业规模已知为A或B或C或D）：公司很小，但未必平稳：Unknown
  - 未知（或者未融资，不需要融资但企业规模未知）：这种企业的发展成谜：Unknown

## 关系

岗位 → REQUIRES → 技能（权重：技能出现频次）

岗位 → OFFERS → 待遇（薪资）

岗位 → BELONGS_TO → 公司

岗位 → LOCATED_IN → 城市 

岗位 → REQUIRES_EDU → 学历要求

岗位 → REQUIRES_EXP → 经验要求

岗位 → TYPE_IS → 岗位名称

岗位名称 → IN_INDUSTRY → 行业

公司 → HAS_SIZE → 企业规模

公司 → HAS_TYPE → 企业类型

城市 → HAS_CLIMATE → 气候区

城市 → HAS_TIER → 城市等级
